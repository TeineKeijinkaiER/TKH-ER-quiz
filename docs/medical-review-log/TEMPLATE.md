# 医学的検証ログ テンプレート

このディレクトリに `YYYY-MM-DD-<対象ファイル名>.md` として保存する。

**使い方:**
1. このテンプレートをコピー
2. 日付・対象ファイル・使用 AI を記入
3. 検証で見つかった指摘を 1 項目ずつ書く
4. 修正したら「対応」を「修正済み（commit X）」に更新

---

## 検証メタデータ

| 項目 | 内容 |
|---|---|
| 検証日 | YYYY-MM-DD |
| 対象ファイル | `data/questions/ja/<level>/<file>.json` |
| 対象 ID 数 | 例: 全 10 問 |
| 使用 AI | 例: Claude Opus 4.7 / ChatGPT GPT-5 / Gemini 2.5 Pro |
| 検証者 | （人間レビュワー名・所属） |
| Web 検索利用 | 利用した / 利用していない |

## 参照したガイドライン・文献
